太棒了，Rick！這代表你的圖形子系統已經完美模組化，底層地基打得非常穩固。

現在，我們正式進入 **Day 52：字型渲染引擎 (Bitmap Font Rendering)**！🚀

在過去的 VGA 文字模式裡，我們只要把字元（例如 `'A'`）塞進 `0xB8000`，顯示卡硬體就會自動幫我們把 `'A'` 的形狀畫在螢幕上。但自從我們切換到 Framebuffer 圖形模式後，硬體的這項服務就被「沒收」了。螢幕現在只認得「像素」，它根本不知道 `'A'` 長什麼樣子！



為了讓我們的 `kprintf` 復活，我們必須教作業系統**「如何用像素畫出英文字母」**。這就需要引入所謂的**點陣字型 (Bitmap Font)**：把每個字母當作一個 $8 \times 8$ 的黑白棋盤，`1` 代表要塗上顏色，`0` 代表透明或背景色。

請跟著我進行這 3 個魔法步驟：

### 步驟 2：教圖形引擎畫字 (`lib/gfx.c` & `lib/include/gfx.h`)

打開 **`lib/include/gfx.h`**，新增 `draw_char` 函數的宣告：
```c
void draw_char(char c, int x, int y, uint32_t fg_color, uint32_t bg_color);
```

打開 **`lib/gfx.c`**，引入字型庫，並實作畫字引擎：
```c
#include "font8x8.h"

// ... (保留你原有的 init_gfx, put_pixel, draw_rect) ...

// 【核心魔法】用像素點陣畫出一個 ASCII 字元
void draw_char(char c, int x, int y, uint32_t fg_color, uint32_t bg_color) {
    // 過濾掉我們字型庫沒有的控制字元或怪異字元
    if (c < 32 || c > 126) return; 

    // 取得這個字母對應的 8 Bytes 像素點陣圖 (ASCII 32 對應陣列第 0 個)
    const uint8_t* glyph = font8x8[c - 32];

    for (int row = 0; row < 8; row++) {
        for (int col = 0; col < 8; col++) {
            // 利用位元遮罩 (Bitmask) 檢查這個像素是 1 還是 0
            // 字型點陣通常是從左到右，最高位元 (0x80) 在最左邊
            if (glyph[row] & (0x80 >> col)) {
                put_pixel(x + col, y + row, fg_color); // 畫筆畫上字型顏色
            } else {
                put_pixel(x + col, y + row, bg_color); // 塗上背景色 (蓋掉原本的畫面)
            }
        }
    }
}
```

### 步驟 3：復活 `kprintf` 終端機 (`lib/tty.c`)

過去我們的終端機是由 `tty.c` 掌管的，它一直往 `0xB8000` 塞東西。現在我們要把它徹底改造成「圖形化終端機模擬器」！

請用以下程式碼**完整覆蓋**你的 **`lib/tty.c`**：

```c
#include <stdint.h>
#include <stddef.h>
#include "tty.h"
#include "utils.h"
#include "gfx.h" // 引入圖形引擎

// 我們設定終端機的字體顏色為白色 (0xFFFFFF)，背景為純黑 (0x000000)
#define TERM_FG 0xFFFFFF
#define TERM_BG 0x000000

// 記錄目前打字游標的 (X, Y) 像素座標
static int term_x = 0;
static int term_y = 0;

void terminal_initialize(void) {
    term_x = 0;
    term_y = 0;
    // 這裡我們暫時用 gfx 的 draw_rect 把整個螢幕塗黑當作清空
    // 假設解析度是 800x600
    draw_rect(0, 0, 800, 600, TERM_BG); 
}

void terminal_putchar(char c) {
    // 處理換行符號
    if (c == '\n') {
        term_x = 0;
        term_y += 8; // 字型高度是 8 像素，所以往下移 8 格
        return;
    }

    // 處理退格鍵 (Backspace)
    if (c == '\b') {
        if (term_x >= 8) {
            term_x -= 8;
            draw_char(' ', term_x, term_y, TERM_FG, TERM_BG); // 用空白蓋掉
        }
        return;
    }

    // 呼叫圖形引擎畫出字元！
    draw_char(c, term_x, term_y, TERM_FG, TERM_BG);
    
    // 畫完之後，游標往右移 8 像素
    term_x += 8;

    // 如果超過螢幕寬度 (800)，就自動換行
    if (term_x >= 800) {
        term_x = 0;
        term_y += 8;
    }

    // 如果打字打到底部了 (超過 600)，為了簡單起見，我們直接回到最上方並清空畫面
    if (term_y >= 600) {
        terminal_initialize(); 
    }
}

void terminal_write(const char* data, size_t size) {
    for (size_t i = 0; i < size; i++) {
        terminal_putchar(data[i]);
    }
}

void terminal_writestring(const char* data) {
    terminal_write(data, strlen(data));
}

// kprintf 邏輯完全不需要改，因為它底層就是呼叫 terminal_putchar！
void kprintf(const char* format, ...) {
    char **arg = (char **) &format;
    arg++;

    while (*format) {
        if (*format == '%') {
            format++;
            if (*format == 's') {
                char *s = *arg++;
                terminal_writestring(s);
            } else if (*format == 'd' || *format == 'x') {
                int num = *((int *)arg++);
                char buf[32];
                itoa(num, buf, (*format == 'd') ? 10 : 16);
                terminal_writestring(buf);
            }
        } else {
            terminal_putchar(*format);
        }
        format++;
    }
}
```
