* **傳輸層**：拆解 UDP、手刻最複雜的 TCP 三方交握與狀態機、處理 Big/Little-Endian 轉換。
* **應用層**：實作 DNS 封包解析、組裝 HTTP GET 請求、剝除 HTTP Headers。
* **系統整合**：將網路堆疊與 VFS (虛擬檔案系統) 完美串接，打通 `SYS_WRITE` 系統呼叫！
